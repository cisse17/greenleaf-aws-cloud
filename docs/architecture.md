## Répartition des rôles

- Chef de Projet / Architecte Cloud :
  - Conception de l'architecture
  - Coordination de l'équipe
  - Rédaction du DAT
  - Validation des choix techniques

- Ingénieur DevOps (Terraform) :
  - Provisionnement AWS avec Terraform
  - VPC, EC2, ALB, Auto Scaling, RDS

- Ingénieur DevOps (Ansible / Magento) :
  - Configuration des serveurs
  - Installation de Magento
  - Automatisation complète
  
- FinOps & Monitoring :
  - Estimation des coûts AWS
  - Mise en place d’AWS Budgets
  - CloudWatch & alarmes
